# Spreadsheet Sample - Ralphループ検証プロジェクト

kiro-cliとRalphループによる自律開発の検証用スプレッドシートアプリケーション

## プロジェクト概要

このプロジェクトは、AIエージェントによる自律開発手法「Ralphループ」の実用性を検証するために開発されました。
コンテキスト腐敗を回避しながら、複雑なアプリケーションを自律的に構築できるかを実証しています。

## Ralphループとは

1つのタスクが終わるごとにコンテキストを破棄し、新しいセッションで次のタスクを開始するループ構造。
長時間のAI実行でもコンテキスト劣化による精度低下を防ぎます。

## 検証結果

- **テスト総数**: 126テスト（ユニット101 + プロパティ25）
- **テストカバレッジ**: 91.5%（Engine層）
- **すべてのテストが自動生成され、パス**

## 技術スタック

- React 19 + TypeScript
- Vite + Vitest
- fast-check（プロパティベーステスト）
- Tailwind CSS

## 実装された機能

- 10列×20行のグリッド
- 数式計算（四則演算、SUM、AVG）
- セル参照と依存関係の自動再計算
- 循環参照検出とエラー処理
